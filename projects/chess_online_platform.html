Technologies Used: React.js, ChakraUI, Auth0, Django, PostgreSQL, Redis, Firebase

  • Implemented real-time multiplayer, dynamic room creation, and Stockfish AI with five difficulty levels and strategic move scoring from -300 to +300.

  • Boosted performance by 27x with Redis caching and ensured robust security with IP blocking and rate limiting.

  • Provided detailed user profiles and match statistics.

  • Deployed on a custom Ubuntu server, showcasing full-stack development and IT infrastructure skills.
